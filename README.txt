Стартовий шаблон проекта з настроїним GULP 1.0.0

1 для початку встановлюємо модулі команда з консолі npm i
2 закидуємо файли по папках (картинки та ін)
3 з консолі запускаємо gulp buildAll(треба запускати коли додаємо шрифти і картинки)
4 запускаємо start_gulp для звичайної роботи
5 для створення нового компонента запускаємо з консолі ./new_component [назва компонента]
(створення html i sass файлів автоматично по папках src/tpl/ i src/sass/blocks/ )
